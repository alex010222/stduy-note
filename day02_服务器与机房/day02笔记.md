# 特殊IP

普通ip地址

分为两段

```
网络地址 + 主机地址
192.168.0.     14
```

> 俩特殊ip
>
> 每一个机器，会有三种ip形式

- 127.0.0.1
- 0.0.0.0
- 具体的网卡地址，192.168.0.140 
  - *网卡可能有多块，就表示，这个机器会有多个IP*
