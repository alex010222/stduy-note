# 友情提醒

数据库方面知识，主要以运维、开发分为两个方向，不同的方向所重点学习的内容不一样

- 运维人员，主要是对数据库架构、设计、维护
  - 单实例、多实例
  - SQL语句基础CURD学习、权限管理
  - 字符集、数据库引擎
  - 备份方案
  - 复制方案
  - 高可用方案
- 开发人员，主要是对数据进行设计、开发
  - 针对业务进行数据库设计、表结构设计
  - 高性能索引
  - 视图
  - 存储过程
  - 函数
  - 等
